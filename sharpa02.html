Titulos

Os títulos definem o contorno do seu site. Existem seis níveis de títulos

Listas Nao Ordenadas

É assim que você faz uma lista não ordenada

  • Adicione um elemento "ul" (que significa lista não ordenada
  • Adicione cada item em seu próprio elemento "li"
  • Eles não precisam estar em nenhuma ordem específica

Listas Ordenadas

É assim que você faz uma lista ordenada

  1. Observe o novo elemento "ol" envolvendo tudo
  2. Mas, os elementos dos itens da lista são os mesmos
  3. Observe também como os números aumentam por si próprios
  4. Você deve estar percebendo que as coisas estão nessa ordem precisa, porque esta é uma lista ordenada
